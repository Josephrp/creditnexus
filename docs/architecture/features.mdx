## Features Overview

CreditNexus provides a comprehensive suite of features for credit agreement processing, verification, compliance, and financial operations. This document provides a high-level overview of all features.

---

## Core Features

### Document Extraction

AI-powered extraction of structured CDM data from unstructured PDF credit agreements.

**Key Capabilities**:
- Multi-modal input (PDF, text, audio, image)
- LLM-based extraction (OpenAI, vLLM, HuggingFace)
- CDM-compliant structured output
- Policy validation at extraction point

**Documentation**: [Document Extraction Feature](/features/document-extraction)

---

### Trade Execution

LMA trade confirmation and settlement with CDM compliance.

**Key Capabilities**:
- Pre-filled trade tickets from FDC3 context
- Policy evaluation before execution
- x402 payment integration
- CDM event generation

**Documentation**: [Trade Execution Feature](/features/trade-execution)

---

### Verification

Satellite-based ground truth verification for sustainability-linked loans.

**Key Capabilities**:
- Sentinel-2 satellite imagery
- NDVI analysis
- TorchGeo ResNet-50 land use classification
- Compliance/breach determination

**Documentation**: [Verification Feature](/features/verification)

---

### Document Generation

LMA template generation from CDM data with AI assistance.

**Key Capabilities**:
- Template-based generation
- AI-powered field population
- Decision support chatbot
- Multi-input support

**Documentation**: [Document Generation Feature](/features/document-generation)

**Key Capabilities**:
