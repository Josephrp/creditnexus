## Overview

The Verification workflow allows you to verify sustainability-linked loans and bonds using satellite imagery and ground truth data.

## Step-by-Step Process

### 1. Access Verification Tools

Navigate to:
- **Green Lens**: ESG visualization and verification
- **Verification Demo**: Ground truth verification
- **Ground Truth**: Detailed verification data

### 2. Upload Loan/Bond Information

Enter or upload:
- Loan details
- Geographic coordinates
- Sustainability claims
- Bond information

### 3. Satellite Analysis

The system performs:
- **NDVI Analysis**: Vegetation index calculation
- **Land Use Classification**: TorchGeo ResNet-50 analysis
- **Change Detection**: Temporal comparison
- **Ground Truth Verification**: Satellite vs. claims

### 4. Review Results

Review verification results:
- NDVI scores and trends
- Land use classifications
- Compliance status
- Risk assessments

### 5. Generate Reports

Create verification reports:
- PDF export
- Data visualization
- Compliance certificates
- Audit trails

## Verification Types

### Sustainability-Linked Loans

Verify ESG claims:
- Green project verification
- Environmental impact assessment
- Sustainability metrics tracking

### Bond Verification

Verify bond terms:
- Credit rating validation
- Collateral verification
- Compliance checking

## Satellite Analysis

### NDVI (Normalized Difference Vegetation Index)

Measures vegetation health:
- Range: -1 to +1
- Higher values indicate healthier vegetation
- Temporal tracking for trends

### Land Use Classification

Identifies land use types:
- Agricultural
- Forest
- Urban
- Water bodies

## Best Practices

1. **Accurate Coordinates**: Ensure precise geographic data
2. **Regular Updates**: Schedule periodic verification
3. **Document Claims**: Maintain detailed documentation
4. **Review Results**: Carefully review verification outputs

## Troubleshooting

**Verification fails:**
- Check coordinate accuracy
- Verify satellite data availability
- Review input data quality

**Unexpected results:**
- Review ground truth data
- Check classification accuracy
- Consult verification team
